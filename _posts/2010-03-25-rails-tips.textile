h2. 覆盖ActiveRecord属性读写方法

b2471adc
比如需要将属性photo的值进行Base64解码后再写入数据库中，可重写方法

{% highlight ruby %}
def photo=(encoded_photo)
  write_attribute(:photo, Base64.decode64(encoded_photo))
end

def photo
  Base64.encode64(read_attribute(:photo))
end
{% endhighlight %}
